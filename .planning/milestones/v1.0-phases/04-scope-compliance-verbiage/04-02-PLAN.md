**Props:** `{ meta: ScopeMeta }`

**Pill base class:** `'inline-flex items-center px-2 py-0.5 rounded-full text-xs font-medium mr-2'` (same as LegalMetaBadge)

**Branch 1: scope-of-work** (`meta.passType === 'scope-of-work'`)
- Pill for `scopeItemType`:
  - 'exclusion' or 'gap' = `bg-red-100 text-red-700`
  - 'ambiguity' = `bg-amber-100 text-amber-700`
  - 'inclusion' or 'scope-rule' = `bg-green-100 text-green-700`
  - 'specification-reference' = `bg-blue-100 text-blue-700`
- Display the label as the scopeItemType value with first letter capitalized
- If `specificationReference` is non-empty and not "N/A", show a second pill in `bg-slate-100 text-slate-600` with the spec reference text
- If `affectedTrade` is non-empty and not "glazing" (the obvious case), show a pill in `bg-purple-100 text-purple-700` with the trade name

**Branch 2: dates-deadlines** (`meta.passType === 'dates-deadlines'`)
- Pill for `periodType`:
  - 'notice-period' or 'cure-period' = `bg-amber-100 text-amber-700`
  - 'payment-term' = `bg-green-100 text-green-700`
  - 'project-milestone' or 'submittal-deadline' = `bg-blue-100 text-blue-700`
  - 'warranty-period' or 'closeout-deadline' = `bg-slate-100 text-slate-600`
- Format periodType for display: replace hyphens with spaces, capitalize first letter of each word
- If `duration` is non-empty and not "N/A", show a second pill in `bg-slate-100 text-slate-700` showing the duration value (e.g., "30 days")
- If `triggerEvent` is non-empty and not "N/A", show a small text span (not pill) in `text-xs text-slate-500 italic` with "Triggered by: {triggerEvent}"

**Branch 3: verbiage** (`meta.passType === 'verbiage'`)
- Pill for `issueType`:
  - 'one-sided-term' or 'overreach-clause' = `bg-red-100 text-red-700`
  - 'missing-protection' = `bg-amber-100 text-amber-700`
  - 'ambiguous-language' or 'undefined-term' = `bg-yellow-100 text-yellow-700`
- Format issueType for display: replace hyphens with spaces, capitalize first letter of each word
- Pill for `affectedParty`:
  - 'subcontractor' = `bg-red-100 text-red-700` with label "Affects Sub"
  - 'general-contractor' = `bg-blue-100 text-blue-700` with label "Affects GC"
  - 'both' = `bg-purple-100 text-purple-700` with label "Affects Both"
  - 'unspecified' = do not render
- If `suggestedClarification` is non-empty and not "N/A", render a small block below the pills:
  ```
  <div className="mt-1 text-xs text-slate-600 italic">
    Suggested: {meta.suggestedClarification}
  </div>
  ```

**Branch 4: labor-compliance** (`meta.passType === 'labor-compliance'`)
- Pill for `requirementType`:
  - 'prevailing-wage' or 'certified-payroll' = `bg-red-100 text-red-700` (legally critical)
  - 'safety-training' or 'licensing' = `bg-amber-100 text-amber-700`
  - All others = `bg-slate-100 text-slate-600`
- Format requirementType for display: replace hyphens with spaces, capitalize first letter of each word
- If `responsibleParty` is non-empty and not "Not specified", show pill in `bg-blue-100 text-blue-700` with the party name
- If `deadline` is non-empty and not "Not specified" and not "N/A", show pill in `bg-amber-100 text-amber-700` with "Due: {deadline}"
- **Checklist rendering:** If `checklistItems` array has length > 0, render a checklist section below the pills:
  ```html
  <div className="mt-2 border-t border-slate-100 pt-2">
    <p className="text-xs font-semibold text-slate-500 uppercase tracking-wide mb-1">Compliance Checklist</p>
    <ul className="space-y-1">
      {checklistItems.map((item, i) => (
        <li key={i} className="flex items-start gap-2 text-xs">
          <span className={`mt-0.5 w-2 h-2 rounded-full flex-shrink-0 ${
            item.status === 'required' ? 'bg-red-400' :
            item.status === 'conditional' ? 'bg-amber-400' : 'bg-green-400'
          }`} />
          <div>
            <span className="text-slate-700">{item.item}</span>
            {item.deadline && item.deadline !== 'Not specified' && (
              <span className="text-slate-400 ml-1">- Due: {item.deadline}</span>
            )}
            {item.responsibleParty && item.responsibleParty !== 'Not specified' && (
              <span className="text-slate-400 ml-1">({item.responsibleParty})</span>
            )}
          </div>
        </li>
      ))}
    </ul>
  </div>
  ```
