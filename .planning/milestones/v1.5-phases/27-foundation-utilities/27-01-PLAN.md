**src/utils/palette.ts:**
- Export `SEVERITY_BADGE_COLORS: Record<Severity, string>` mapping each severity to complete Tailwind class strings: Critical='bg-red-100 text-red-700 border-red-200', High='bg-amber-100 text-amber-700 border-amber-200', Medium='bg-yellow-100 text-yellow-700 border-yellow-200', Low='bg-blue-100 text-blue-700 border-blue-200', Info='bg-slate-100 text-slate-700 border-slate-200'
- Export `getRiskScoreColor(score: number): string` — returns 'text-red-600' if >70, 'text-amber-600' if >40, 'text-emerald-600' otherwise
- Export `getRiskBadgeColor(score: number): string` — returns 'bg-red-100 text-red-700' if >=70, 'bg-amber-100 text-amber-700' if >=40, 'bg-emerald-100 text-emerald-700' otherwise
- Import Severity type from '../types/contract'
- Use ONLY complete string literals (no interpolation) to survive Tailwind JIT purge

**src/utils/errors.ts:**
- Export type `ErrorType = 'network' | 'api' | 'validation' | 'storage' | 'timeout' | 'unknown'`
- Export interface `ClassifiedError { type: ErrorType; userMessage: string; retryable: boolean; originalError: unknown }`
- Export interface `ApiErrorResponse { error: string; type: ErrorType; retryable: boolean; details?: unknown }`
- Export function `classifyError(err: unknown): ClassifiedError` with detection for:
  - Network: TypeError 'Failed to fetch' or message includes 'NetworkError' — retryable
  - Timeout: message includes 'HeadersTimeoutError', 'timeout', or 'ETIMEDOUT' — retryable
  - Storage: DOMException with QuotaExceededError or NS_ERROR_DOM_QUOTA_REACHED — not retryable
  - API: object with numeric `status` field (429=retryable, 401/others=not retryable)
  - Unknown: fallback with err.message if Error, else generic message — not retryable
- Export function `formatApiError(classified: ClassifiedError, details?: unknown): ApiErrorResponse`
- NO React imports, NO DOM imports — must be isomorphic (client + server)

**src/storage/storageManager.ts:**
- Define `StorageRegistry` interface mapping all 5 keys to their value types:
  - `'clearcontract:contracts'`: Contract[] (from '../types/contract')
  - `'clearcontract:contracts-seeded'`: string
  - `'clearcontract:schema-version'`: string
  - `'clearcontract:company-profile'`: CompanyProfile (from '../knowledge/types')
  - `'clearcontract:hide-resolved'`: string
- Export type `StorageKey = keyof StorageRegistry`
- Export interface `StorageResult<T> { ok: boolean; data: T | null; error: string | null; quotaExceeded: boolean }`
- Export `load<K>(key: K): StorageResult<StorageRegistry[K]>` — reads from localStorage, JSON.parse, returns typed result. Null raw value = ok with null data. Parse error = not ok.
- Export `save<K>(key: K, value: StorageRegistry[K]): StorageResult<null>` — JSON.stringify + setItem. Catches DOMException for quota.
- Export `loadRaw<K>(key: K): StorageResult<string>` — reads raw string without JSON.parse. For keys that store plain strings (seeded, schema-version, hide-resolved).
- Export `saveRaw<K>(key: K, value: string): StorageResult<null>` — writes raw string without JSON.stringify. Catches DOMException for quota.
- Export `remove<K>(key: K): void` — removeItem with try-catch swallow.
- NO React imports — pure utility.
