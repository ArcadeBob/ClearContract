## Cross-Phase Integration Gaps

### CRITICAL: All Contracts View Unreachable (Phase 15)

**File:** `src/hooks/useRouter.ts` lines 50-53
**Issue:** `navigateTo('contracts')` falls into the else branch which remaps to `'dashboard'` and pushes URL `/`. The AllContracts page is dead code.
**Affected:** ROUTE-01 (back/forward never reaches contracts), ROUTE-02 (no URL for contracts view)
**Fix:** Add `contracts` case in `navigateTo` and `/contracts` path in `parseUrl`

### MODERATE: CSV Export Ignores Filters (Phase 19 x Phase 16)

**File:** `src/pages/ContractReview.tsx` line 225
**Issue:** `exportContractCsv(contract)` exports all findings. The `hideResolved` filter and `selectedCategory` filter from Phase 16 are not applied.
**Affected:** EXPORT-02
**Fix:** Pass `visibleFindings` to `exportContractCsv` instead of full contract
