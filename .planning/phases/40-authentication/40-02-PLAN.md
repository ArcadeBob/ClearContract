**2. Modify `src/App.tsx`** -- Add auth gate at top of component:

Add imports:
```typescript
import { useAuth } from './contexts/AuthContext';
import { LoginPage } from './pages/LoginPage';
import { LoadingScreen } from './components/LoadingScreen';
```

Inside the `App` function body, BEFORE all existing hooks and logic, add the auth gate:
```typescript
const { session, isLoading, signOut } = useAuth();

if (isLoading) {
  return <LoadingScreen />;
}

if (!session) {
  return <LoginPage />;
}
```

This means:
- When `isLoading` is true: render ONLY LoadingScreen (no sidebar, no routes) -- AUTH-06
- When `!session`: render ONLY LoginPage (no sidebar, no routes) -- AUTH-03
- When session exists: render the full existing app (sidebar + content) -- AUTH-01/02

Pass `signOut` to Sidebar. Update the Sidebar JSX to add the `onSignOut` prop:
```tsx
<Sidebar
  activeView={activeView}
  onNavigate={(view) => navigateTo(view)}
  contractCount={contracts.length}
  onSignOut={signOut}
/>
```
