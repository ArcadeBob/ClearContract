## Interaction States

### Sign In Button States

| State | Visual |
|-------|--------|
| Default | bg-slate-900, white text, "Sign In" label |
| Hover | bg-slate-800 |
| Submitting | opacity-50, cursor-not-allowed, disabled. Label replaced with CSS spinner (w-5 h-5 border-2 border-white/30 border-t-white rounded-full animate-spin) centered in button |
| Disabled (empty fields) | Same as submitting visual -- opacity-50, cursor-not-allowed |

### Input Field States

| State | Visual |
|-------|--------|
| Default | border-slate-300, bg-white |
| Focus | border-blue-500, ring-2 ring-blue-500/20 |
| Error visible | No change to input styling (error is shown as text below, not on the input itself) |

### Auth Flow States

| State | What Renders | Duration |
|-------|-------------|----------|
| Initial load (session unknown) | LoadingScreen | Until `INITIAL_SESSION` event fires (typically < 500ms) |
| Authenticated | Full app (Sidebar + page content) | Persistent |
| Unauthenticated | LoginPage only (no sidebar, no nav, no routes) | Until successful sign-in |
| Signing in | LoginPage with spinner button | Until auth response |
| Sign-in error | LoginPage with inline error message | Until user modifies a field |
| Signing out | Immediate transition to LoginPage | Instant |

## Accessibility Notes

| Concern | Implementation |
|---------|---------------|
| Form labels | Each input has a visible `<label>` element with `htmlFor` attribute matching input `id` |
| Error announcement | Error message container uses `role="alert"` for screen reader announcement |
| Focus management | On login page mount, focus the email input. After error, keep focus on the form (do not reset to top) |
| Button loading state | Submit button includes `aria-disabled="true"` when submitting; spinner has `aria-hidden="true"` |
| Keyboard | Form submits on Enter key (native form behavior). Tab order: email -> password -> sign in button |
| Color contrast | Error text red-600 on white/80 background: 4.6:1 ratio (passes WCAG AA). Slate-900 button text white: 15.4:1 ratio |
