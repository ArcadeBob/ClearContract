## Interaction States

### Upload Page State Machine

```
IDLE (UploadZone visible)
  |-- user drops/selects PDF --> ANALYZING

ANALYZING (spinner visible, UploadZone hidden)
  |-- analysis succeeds + user still on page --> navigate to /review/{id}
  |-- analysis succeeds + user navigated away --> success toast with "View" action
  |-- analysis fails --> ERROR_TOAST (stay on current page)
  |-- user clicks "Back" during analysis --> navigates away, analysis continues in background

ERROR_TOAST
  |-- user clicks "Retry" --> ANALYZING (re-triggers upload)
  |-- toast auto-dismisses (non-retryable) --> IDLE
```

### Key Behavioral Changes from Current

| Before (Phase 42) | After (Phase 43) |
|-------------------|------------------|
| Upload triggers immediate navigation to review page with placeholder contract | Upload stays on upload page showing analyzing spinner |
| Placeholder "Analyzing" contract visible in review page | No placeholder contract exists -- contract only appears after server returns it |
| Cancel analysis deletes placeholder contract | Cancel analysis is just navigation away; no cleanup needed |
| analyzeContract sends companyProfile in request body | analyzeContract sends only pdfBase64 + fileName; auth token in header |

## Analyzing Indicator Spec

The analyzing indicator replaces the UploadZone within the existing card container on the ContractUpload page.

**Layout:** Vertically centered within the white card (same card that houses UploadZone). Flex column, items-center, text-center.

**Spinner:** Lucide `Loader2` icon at 32x32px (w-8 h-8), text-blue-600, with CSS `animate-spin` (Tailwind built-in). Wrapped in a bg-blue-100 rounded-full container at 64x64px (p-4) matching the existing UploadCloud icon container pattern.

**Text stack:**
- Heading: "Analyzing Contract..." -- text-lg font-semibold text-slate-900 mb-2 (matches UploadZone heading style)
- Body: "AI is reviewing your contract. This usually takes 30-60 seconds." -- text-slate-500 text-sm max-w-xs mx-auto mb-2
- Subtext: "You can navigate away -- we'll notify you when it's ready." -- text-slate-400 text-xs

**Animation:** Framer Motion `motion.div` with `initial={{ opacity: 0 }}`, `animate={{ opacity: 1 }}`, `transition={{ duration: 0.3 }}`. Use `AnimatePresence` to crossfade between UploadZone and analyzing indicator.

**Accessibility:** Spinner container has `role="status"` and `aria-label="Analyzing contract"`. The `Loader2` icon has `aria-hidden="true"`.
