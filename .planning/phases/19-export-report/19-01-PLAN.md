  <action>
Create `src/utils/exportContractCsv.ts` with three exported functions:

1. `escapeCsv(value: string | undefined | boolean): string` -- RFC 4180 compliant field escaping:
   - Convert undefined/null to empty string, boolean to "Yes"/"No", everything else to String
   - If the string contains comma, double-quote, newline, or carriage return, wrap in double quotes and double any existing double quotes
   - Return the string as-is otherwise

2. `exportContractCsv(contract: Contract): string` -- Generates the full CSV content string:
   - **Metadata header rows** (key-value pairs, one per row):
     - Contract Name, contract.name
     - Contract Type, contract.type
     - Risk Score, contract.riskScore (as string)
     - Bid Signal, `{contract.bidSignal.label} ({contract.bidSignal.score})` -- only if bidSignal exists
     - Analysis Date, contract.uploadDate
     - Total Findings, contract.findings.length (as string)
   - **Blank row** separator
   - **Column headers row**: Severity, Category, Clause Reference, Clause Text, Explanation, Recommendation, Negotiation Position, Resolved, User Note
   - **Finding data rows**: ALL findings from contract.findings (no filtering), sorted by severity using rank order Critical=0, High=1, Medium=2, Low=3, Info=4. For each finding, use nullish coalescing (`?? ''`) on all optional fields: clauseReference, clauseText, explanation, recommendation, negotiationPosition, note. Use `f.resolved ?? false` for the Resolved column (outputs "Yes"/"No").
   - **Blank row** separator (only if contract.dates.length > 0)
   - **Key Dates header row**: "Key Dates" in first column
   - **Key Dates column headers**: Date, Description, Type
   - **Key Dates data rows**: Each date from contract.dates with d.date, d.label, d.type
   - Join all lines with `\r\n` (Windows-style line endings for Excel compatibility)

3. `downloadCsv(content: string, filename: string): void` -- Triggers browser download:
   - Create a Blob with UTF-8 BOM prefix (`\uFEFF`) prepended to content, MIME type `text/csv;charset=utf-8;`
   - Create object URL via `URL.createObjectURL`
   - Create temporary anchor element, set href and download attribute, append to body, click, remove from body
   - Revoke the object URL via `URL.revokeObjectURL`

4. `sanitizeFilename(name: string): string` -- Cleans contract name for filesystem:
   - Remove all characters except alphanumeric, spaces, and hyphens
   - Replace whitespace sequences with single hyphen
   - Collapse multiple hyphens to single hyphen
   - Trim leading/trailing hyphens

Import `Contract` and `Severity` types from `../types/contract`.

Use a const `SEVERITY_ORDER: Severity[]` = ['Critical', 'High', 'Medium', 'Low', 'Info'] for sort ranking.

Use a helper `csvRow(fields: (string | undefined | boolean)[]): string` that maps fields through escapeCsv and joins with commas.
