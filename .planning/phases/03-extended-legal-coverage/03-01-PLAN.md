**Part B: Add 7 new passes to ANALYSIS_PASSES array** -- Append after the existing legal-retainage pass. Each pass follows the established pattern: { name, isOverview: false, isLegal: true, schema, systemPrompt, userPrompt }.

**Pass 1: legal-insurance (LEGAL-06)**
System prompt must include:
- Role: glazing/glass installation subcontractor perspective
- Task: Find and analyze ALL insurance requirements in the contract
- Two-part output: (1) One summary checklist finding listing ALL coverage requirements with coverageItems and endorsements populated, (2) Individual findings for each gap/unusual requirement at Critical/High severity
- Coverage standards for glazing subs: CGL $1M/$2M (industry standard), commercial umbrella $2M-$5M, auto liability $1M CSL, workers comp statutory, builder's risk/installation floater project-dependent
- Standard endorsements: additional insured (CG 20 10, CG 20 37), waiver of subrogation, primary/non-contributory
- Severity rules: CGL above $1M/$2M = High, non-standard hard-to-obtain endorsements = High, missing insurance section = Medium
- MUST search ENTIRE contract (insurance reqs may appear in general conditions, exhibits, attachments, flow-down provisions)
- Populate coverageItems and endorsements arrays fully on the summary finding
- Certificate holder details in certificateHolder field
- If no insurance requirements found, produce a Medium finding noting their absence
User prompt: "Analyze all insurance requirements in this glazing subcontract. Produce a summary checklist and individual findings for gaps or unusual requirements."

**Pass 2: legal-termination (LEGAL-07)**
System prompt:
- Find ALL termination clauses (for-convenience, for-cause, mutual)
- Severity rules (MANDATORY): Termination for convenience with no compensation = Critical. Termination for convenience with partial payment = High. For-cause with short cure period (<7 days) = High. Standard for-cause with reasonable cure period = Medium.
- Extract: terminationType, noticePeriod, compensation, curePeriod
- Flag if only one party has termination rights (one-sided)
- Flag missing cure period provisions
- Verbatim quoting, glazing sub perspective, state-specific context when available
User prompt: "Analyze all termination clauses in this glazing subcontract."

**Pass 3: legal-flow-down (LEGAL-08)**
System prompt:
- Find and analyze ALL flow-down provisions
- Severity rules: Blanket flow-down of ALL prime contract terms = Critical. Flow-down imposing obligations beyond sub's scope or insurance = High. Flow-down of specific relevant sections = Medium. Targeted flow-down with exceptions = Low.
- When flow-down references a prime contract the sub hasn't seen: create a High finding noting the sub is bound by unseen terms, recommend requesting the prime contract
- Trace specific problematic obligations that flow down: schedule penalties, warranty periods, insurance beyond sub's coverage, indemnification broader than negotiated
- Populate problematicObligations array with the dangerous obligations
- primeContractAvailable = false when the contract incorporates prime terms by reference without making them available
- crossReferences should reference CONTRACT SECTIONS that interact with the flow-down clause (e.g., "Section 12.3 - Insurance"), NOT other analysis findings
- Focus on dangerous obligations, not every single one
User prompt: "Analyze all flow-down provisions in this glazing subcontract."

**Pass 4: legal-no-damage-delay (LEGAL-09)**
System prompt:
- Find ALL no-damage-for-delay clauses
- Severity rules: Absolute waiver of ALL delay claims = Critical. Broad waiver with narrow exceptions = High. Waiver with reasonable exceptions (owner-caused delays, force majeure) = Medium.
- Populate waiverScope enum accurately
- List all exceptions in the exceptions array
- Provide enforceabilityContext: when governing law state is known, note whether state prohibits/limits no-damage-for-delay clauses. Key states: CO, WA, OR, VA (unreasonable delays), OH, NC, LA, MO (public), NJ (negligence/bad faith), IN (unforeseen). Even in enforcing states, note judicial exceptions (active interference, bad faith, unreasonable delays).
- Provide general guidance on deadlines. Always recommend consulting a construction attorney for precise deadlines.
User prompt: "Analyze all no-damage-for-delay clauses in this glazing subcontract."

**Pass 5: legal-lien-rights (LEGAL-10)**
System prompt:
- Find ALL provisions affecting the sub's mechanic's lien rights
- Focus on payment-contingent waivers: unconditional waivers required BEFORE payment, waivers of retainage lien rights, broad release language waiving rights beyond the specific payment
- Severity rules: No-lien clause or unconditional waiver before payment = Critical. Broad release provisions waiving lien rights = High. Conditional waiver tied to actual payment = Low. Missing lien rights provisions = Medium (flag as gap).
- Always flag no-lien clauses even when potentially unenforceable -- include state-specific enforceability context
- When governing law state is known, include statutory lien filing deadline with caveat: "Exact deadlines may vary based on project type, contractor tier, and notice filings. Consult a construction attorney for precise deadlines."
- Populate waiverType, lienFilingDeadline, enforceabilityContext
User prompt: "Analyze all provisions affecting lien rights in this glazing subcontract."

**Pass 6: legal-dispute-resolution (LEGAL-11)**
System prompt:
- Find ALL dispute resolution provisions
- Combined venue + attorney fee analysis
- Severity rules: Mandatory arbitration with no appeal + distant venue = Critical. Mandatory arbitration in reasonable venue = High. Litigation with unfavorable venue/jurisdiction = High. One-sided attorney fee clause (loser-pays or sub-only) = Critical. No fee shifting = Medium. Mutual fee shifting = Low. Mediation first, then arbitration or litigation = Medium.
- Populate mechanism, venue, feeShifting, mediationRequired
- Explain impact from glazing sub perspective -- travel costs, upfront arbitration fees, appeal limitations
User prompt: "Analyze all dispute resolution provisions in this glazing subcontract."

**Pass 7: legal-change-order (LEGAL-12)**
System prompt:
- Find ALL change order and change directive provisions
- Combined protection + notice requirements analysis
- Severity rules: Unilateral change rights with no price adjustment = Critical. Proceed-pending with no payment guarantee = Critical. No written CO required (oral changes binding) = Critical. Unilateral changes with price adjustment mechanism = High. Written CO required but unreasonable timeline = High. Standard mutual change order process = Low.
- Populate changeType, noticeRequired, pricingMechanism, proceedPending
- Flag unreasonable notice timelines (< 3 days)
- Explain proceed-pending risk from glazing sub perspective -- performing work at risk before approval
User prompt: "Analyze all change order and change directive provisions in this glazing subcontract."
